TITLE: running your own cyber-dojo server on Linux

install docker

$ sudo curl -sSL https://get.docker.com/ | sh

install the cyber-dojo shell script

$ curl -O https://raw.githubusercontent.com/cyber-dojo/commander/master/cyber-dojo $ chmod +x cyber-dojo

bring up your cyber-dojo server

$ sudo ./cyber-dojo up
The first time you run this it will download the necessary docker images for your server.
Put your server's IP address into your browser.
That's it!

pre-pull all the language+test images

Pre-pulling a language+test image speeds up setting up a practice-session since the server does not have to pull it. To pre-pull all the images (this will take a while), type:
$ sudo ./cyber-dojo start-point pull languages

pre-pull an individual language+test image

To display all the language+test image names, type:
$ sudo ./cyber-dojo start-point inspect languages

problems?

  • Some cyberdojo images are docker-version-sensitive.
  • To determine the version of docker running, type:
    $ sudo docker --version
  • If your version of docker is not in this list please upgrade docker or email me.

bring down your cyber-dojo server

$ sudo ./cyber-dojo down
