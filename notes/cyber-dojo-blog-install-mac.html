running your own cyber-dojo server on a Mac

install Docker-Toolbox for Mac

From here.

open a Docker Quickstart Terminal

install the cyber-dojo shell script

In the Docker Quickstart Terminal
$ curl -O https://raw.githubusercontent.com/cyber-dojo/commander/master/cyber-dojo $ chmod +x cyber-dojo

bring up your cyber-dojo server

In the Docker Quickstart Terminal
$ ./cyber-dojo up
The first time you run this it will download the necessary docker images for your server.
This may take several minutes depending on your network connection.

access the server via a browser

In the Docker Quickstart Terminal, find your server's IP address
$ docker-machine ip default
Put your server's IP address into your browser.

bring down your cyber-dojo server

In the Docker Quickstart Terminal
$ ./cyber-dojo down

pre-pulling language images

Each practice session is associated with a given language and test framework (eg C# and NUnit).
If the server has not yet pulled its associated docker-image (eg cyberdojofoundation/csharp_nunit) it will pull it.

problems?

  • Some cyberdojo images are docker-version-sensitive.
  • To determine the version of docker running, in the Docker Quickstart Terminal, type:
    $ docker --version
  • If your version of docker is not in this list please upgrade Docker Toolbox or email me.
